# 口播脱稿练习工具

一款帮助用户从"死记硬背全文"转变为"理解核心、自主复述、自然脱稿"的轻量化口播训练工具。

## 项目需求

### 核心价值
- 帮用户把长文变提纲
- 帮用户建立逻辑理解，而不是机械记忆
- 提供阶梯式脱稿练习，降低表达压力
- 社恐友好、无评价、无压力、私密练习

### 适用人群
- 社恐/内向，需要练习表达、演讲、自我介绍的人
- 只能背稿、无法脱稿、不会提炼重点的人
- 需要反复练习同一内容，逐步脱稿的人

### 功能特性

| 模块 | 功能 |
|--------|------|
| **文稿输入** | 粘贴/输入口播文稿（50～1000字） |
| **AI智能拆解** | 输出：核心主旨 + 关键要点（3-6条）+ 复述提示 |
| **阶梯练习** | 4种模式：原文对照 → 仅提纲 → 半提示 → 完全脱稿 |
| **录音回放** | 一键录音、播放、重录（不打分、不评判） |
| **历史记录** | 保存最近20次练习，支持删除 |

### 练习模式说明

| 模式 | 显示内容 |
|------|----------|
| 原文对照 | 左侧原文 + 右侧AI提纲（双栏布局） |
| 仅提纲 | 核心主旨（大字）+ 关键要点列表 |
| 半提示 | 每段仅显示关键词/首字，其余隐藏 |
| 完全脱稿 | 空白 + 柔和提示"尝试脱稿复述" |

## 安装

### 前置要求
- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装步骤

```bash
# 进入项目目录
cd koubo

# 安装依赖
npm install
```

## 启动

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 生产构建

```bash
npm run build
```

构建产物会输出到 `dist` 目录。

## 使用指南

### 1. 配置 API Key
首次使用需要配置通义千问 API Key：

1. 访问 [阿里云 DashScope 控制台](https://dashscope.console.aliyun.com/)
2. 登录/注册阿里云账号
3. 开通「通义千问」服务
4. 在「API-KEY 管理」中创建新的 API Key
5. 复制 API Key，粘贴到本应用的设置页面

### 2. 开始练习
1. 在首页文本框中粘贴练习文稿（50-1000字）
2. 点击「生成提纲 & 开始练习」
3. AI 会自动拆解出核心主旨和关键要点
4. 进入练习页，选择练习模式

### 3. 切换练习模式
点击顶部 Tab 切换 4 种练习模式，建议顺序：
- 原文对照（理解内容）
- 仅提纲（重点模式）
- 半提示（强迫组织语言）
- 完全脱稿（真正演练）

### 4. 录音回放
- 点击底部录音按钮开始录音
- 录音后可播放对比
- 点击重录按钮重新录制

### 5. 查看历史
首页底部显示最近2条练习记录，点击「查看全部历史记录」进入历史页，可删除单条或清空全部。

## 技术栈

- **框架**: React 18 + Vite
- **样式**: Tailwind CSS
- **路由**: React Router v6
- **AI**: 通义千问 API
- **存储**: localStorage（本地存储，不上传云端）
- **录音**: MediaRecorder API（浏览器原生）

## 隐私说明

- 所有文稿与录音数据仅存储在浏览器 localStorage
- 不会上传到任何服务器
- API Key 仅存储在本地
- 不收集用户个人信息
